Visit the page
=============
http://www.onlinepostit.com

## Getting Started

1. `bundle install`
2. `rake db:drop; rake db:create; rake db:migrate; rake db:seed;`
3. `shotgun -p 3000 -o 0.0.0.0`
4. Visit `http://localhost:3000/` in your browser
